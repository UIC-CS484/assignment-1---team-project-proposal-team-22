Extinct or Alive

What RESTful APIs will be used?

GBIF REST API
This API is the primary source of data. It'll provide almost all required information about each animal species. This data will be used highlight critically endangered species.
Wikimedia REST API
This'll be used to complement the GBIF data, by providing the user with images, short description and other information about the species.

What data will be collected from authenticated users?

Besides the user's email, all that will be stored are the comments written by the user and a list of species that they're interested in

What are the entities and their attributes that'll be stored in the database?

  • User -> email, username, interestedSpecies, commentIDs
  • Species -> id, name, scientificName, population, status, region, image, description
  • Comment -> id, username, text, parentID

Who are the potential users of this website?

People who are already passionate about conservation and rewilding. However, more importantly, this website can be a good starting point anyone who isn't convinced to the scale of the problems faced today. So, it can act as a means for the aforementioned group to spread awareness.

What are the goals that this website hopes to achieve?

  • To spread awareness about the scale at which wildlife is being destroyed.
  • Try to highlight humanity's role in this problem.
  • Provide resources for people who want to help and take action.

Why was this topic selected?

Although, at this point most people are aware of the problems that this planet is facing. I believe most of them are still unaware about how serious the issue is. I wanted to build something would make more people take climate change seriously.
